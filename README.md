# Cracking Contexto

## Installation

```
pip install contextocracker
```

## Documentation

Docs are hosted [here](https://cracking-contexto.readthedocs.io/en/latest/) or can be built locally with `mkdocs`

```
pip install mkdocs mkdocs-material pymdown-extension
mkdocs serve
```
